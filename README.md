# **Projeto: Análise Preditiva de Churn de Clientes**
🎯 **Objetivo**
Este projeto tem como objetivo principal identificar padrões de evasão de clientes (Churn) utilizando técnicas de Ciência de Dados e Machine Learning. A análise transforma dados brutos em insights estratégicos para apoiar a tomada de decisão da equipe de retenção, visando a redução da taxa de perda de clientes e o aumento do Life Time Value (LTV).

📊 **Metodologia**
O projeto foi desenvolvido seguindo o ciclo de vida completo de um modelo preditivo:

**Limpeza e Pré-processamento:** Tratamento de valores ausentes, conversão de tipos de dados e codificação de variáveis categóricas (One-Hot Encoding otimizado).

**Modelagem:** Comparação entre Regressão Logística e Random Forest para encontrar o melhor desempenho.

**Avaliação:** Análise baseada em métricas como F1-Score e Recall para garantir a identificação correta dos clientes propensos ao cancelamento.

**Análise de Importância:** Identificação das variáveis que mais influenciam a decisão do cliente em permanecer ou sair.

🚀 **Principais Insights**
A análise revelou que o Churn é impulsionado principalmente por fatores como:

**Modelo de Contrato:** Clientes com planos mensais apresentam maior risco de evasão.

**Tempo de Relacionamento (Tenure):** Identificamos maior vulnerabilidade nos primeiros meses de contrato.

**Serviços Adicionais:** A correlação entre o tipo de serviço de internet (ex: Fibra Óptica) e a propensão ao cancelamento.

💡# **Estratégias de Retenção Sugeridas**
Com base nos resultados, o plano estratégico inclui:

**Campanhas de Migração:** Incentivo para migração de planos mensais para anuais.

**Onboarding Ativo:** Foco em reforçar o valor do serviço nos primeiros 90 dias de contrato.

**Atendimento Preventivo:** Utilização do modelo preditivo para sinalizar clientes de alto risco ao time de Customer Success.
